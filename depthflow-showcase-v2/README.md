# Depthflow Showcase V2

## Installation

First install [ComfyDock](https://github.com/ComfyDock/ComfyDock) CLI:
```bash
# Via uv (recommended)
uv tool install comfydock-cli  

# Via pip
pip install comfydock-cli      
```

Then import the workflow:
```bash
cfd import https://github.com/akatz-ai/ComfyUI-Akatz-Workflows.git#depthflow-showcase-v2
```
